# Новогодняя гирлянда

## Описание

### Системные требования

- Приложение работает только на Windows 10+
- Приложение может нагружать слабые компьютеры (нагрев, активная работа вентиляторов/кулера)
- Приложение раз в 5 секунд устанавливает «поверх всех окон» для всех гирлянд (окон)

### Приложение

Приложение добавляет на верх всех обнаруженных экранов красивую новогоднюю гирлянду
